css는 html에서 디자인 기능을 뺏은 것
css로 하는 게 훨씬 중요함

헤드 위에 style 태그를 통해 원하는 태그에 효과를 부여할 수 있음
또 태그안에 속성을 넣어서 따로따로 효과를 줄 수도 있음

style=""은 html의 속성 "" 사이에는 css 효과가 들어옴


a {} 를 셀렉터,선택자라고 하고 뒤에 효과를 데컬러레이션라고 한다
효과를 지정하고는 ; 필수
이렇게 하면 같은 줄에 써도 효과가 나옴

선택자에는 3가지 분류가 있음

h1이나 a와 같은 일반선택자(내가 그냥 편하게 부르는 거)
